# OthTips

`OthTips` is a World of Warcraft tooltip addon that replaces Blizzard tooltip styling with a custom lightweight layout.

## Features

- Custom player and NPC tooltip formatting
- Mouse anchor or movable fixed anchor
- AddOn settings panel for anchor, delay, padding, font, and cursor gap
- Bundled Inter font options

## Install

Copy the `OthTips` folder into:

`World of Warcraft\_retail_\Interface\AddOns`

## Development

This repository contains the live addon source for local development and packaging.
